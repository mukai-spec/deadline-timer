# 修論タイマー

ただのタイマーです。

先輩方が修論発表に追われていたので、先輩方を鼓舞するときに使えます。
仲のいい先輩  か  優しい先輩にのみ使ってください。

#### あまり仲良くない先輩に使うと、

#### おそらく、仲が悪くなります。



### 使い方

1. イベント名 と その日付を入力
2. 左下のスタートマークを押す
3. 締め切りまでの時間が表示されます
